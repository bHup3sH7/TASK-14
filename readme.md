🤔 What Did We Learn?
We wrote a JavaScript program that works with numbers — using only loops (like repeating steps) and no shortcuts from the Math library. Think of it like doing calculations by hand instead of using a calculator.
Here's what the program does in simple words:

➕ Sum of first N numbers — Adds numbers from 1 up to any number you pick
📋 Multiplication Table — Prints the table of any number (like school tables!)
🔍 Prime Number Check — Checks if a number has no divisors other than 1 and itself
🧩 Print All Factors — Finds all numbers that divide evenly into it
🔢 Sum of Digits — Breaks a number into digits and adds them (e.g. 153 → 1+5+3 = 9)
🏆 Armstrong Number — Checks a cool property where digit powers add back to the number

▶️ How to Run the File

1. Go to your project folder
2. Double click index.html
3. Opens in browser with CSS styled + JS running ✅
